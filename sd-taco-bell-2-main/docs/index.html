Description

  • pShare is a decentralized, peer-to-peer (P2P) file storage system designed for privacy-conscious users and small enterprises.
  • Our system enables users to securely store files by splitting them into smaller chunks, encrypting them, and distributing them across multiple nodes in the network.
  • Using your spare computers or other users' machines, pShare transforms unused storage into a distributed storage network.
  • The project features robust encryption (AES-256 GCM), and during decryption, file integrity is ensured with a checksum verification process to detect any changes in the files.
  • This approach removes reliance on centralized cloud storage providers and ensures greater security and cost efficiency.

Tools, Libraries, APIs

  • Python
    • Cryptodome: Library used for AES encryption and decryption.
    • hashlib: For generating file hashes (SHA-512) to ensure file integrity through checksums.
    • shutil: For file handling and directory management during file chunking and storage.
    • AWS S3 Boto API: Used for integration with cloud storage services (optional feature).
  • Flask + Jinja
    • Used for the web user interface for easy access and management of the distributed storage system.
